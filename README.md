# PyFuscation

Requires python3 

usage: PyFuscation.py [-h] [-f] [-v] [-p] [--ps SCRIPT] 

Optional arguments: 
-h, --help show this help message and exit 
-f    Obfuscate functions
-v    Obfuscate variables
-p    Obfuscate parameters

Required arguments: 
--ps  SCRIPT Obfuscate PowerShell script
